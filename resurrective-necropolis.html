Resurrective Necropolis

An algorithmic memorial generating names from 80+ languages and cultures worldwide. Each name represents the linguistic diversity of human naming traditions.

Features:

- Language-matched voice synthesis for authentic pronunciation
- Cultural naming conventions (patronymics, honorifics, clan names)
- Special cultural practices (death taboos, sacred names)
- Historical names from various periods
- Name meanings and etymologies
- Gender diversity including cultural third genders

Controls:

- Speed: Slow (8s) or Fast (3s) between names
- Weighted: Names appear proportional to speaker population
- Detailed: Show name components and meanings
- Historical: Prefer historical/ancient names
