## Field Definitions

| Field | Purpose | Max Length |
|-------|---------|------------|
| `name` | Display name | 50 chars |
| `formula` | Key formula/fact | 30 chars |
| `correct` | What's true | 1-2 sentences |
| `error` | What's wrong with this answer | 1 sentence |
| `fix` | Correct way to think | 1 sentence |
| `action` | What to do right now | 1 sentence, imperative |
| `gap` | Underlying misunderstanding | 1 sentence |
| `persistent.action` | Study task | 1-2 sentences |

## Summary

| Principle | Implementation |
|-----------|---------------|
| **Pithy** | Each field is 1-2 sentences max |
| **Deficit-driven** | Focus on `error`, `gap`â€”what's wrong |
| **Actionable** | Every error has an `action` field with imperative verb |
| **No fluff** | No "Great job!", no padding, no repetition |

The feedback tells students:
1. **What you got wrong** (the error)
2. **Why it's wrong** (the fix)
3. **What to do about it** (the action)

Nothing more.
