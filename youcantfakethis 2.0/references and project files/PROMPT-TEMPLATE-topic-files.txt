□ 8 concepts, each with:
  - formula (correct answer explanation)
  - correct (feedback for right answer)

  - Each question has: correct, distractors (3), concept, tooltip

════════════════════════════════════════════════════════════════════════════════
NOTES & LESSONS LEARNED
════════════════════════════════════════════════════════════════════════════════

WHY FILES GET COMPRESSED:
- Asking for "parallel" or "in one render" creation triggers compression
- Claude tries to fit multiple large files into fewer responses
- Result: ~1/3 the intended content size

WHAT PREVENTS COMPRESSION:
- "ONE AT A TIME sequentially" instruction
- "do NOT compress" explicit warning
- Providing a reference file for quality benchmark
- Checking file sizes match the example (~25-35KB per file typically)

QUALITY CHECKPOINTS:
- Each file should be ~25-35KB
- 8 concepts × 3 wrong answers = 24 error/fix/action sets
- 4 tabs × 3 rounds × 3 sentences = 36 sentences minimum
- Every dropdown question needs a tooltip

FILE NAMING CONVENTION:
[Unit]AOS[Number]-[TopicNumber]-[TopicName].json
Example: U3AOS1-07-GeneCloning.json
