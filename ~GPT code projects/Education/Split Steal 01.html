Split/Steal Rules

  • Each player starts with $10
  • Players are randomly paired each round (never same pair twice in a row)
  • Each pair chooses:
    • Both Split: Each keeps their money
    • One Steals: Stealer gets both amounts, other player eliminated
    • Both Steal: Both lose money and are eliminated
  • Lost funds (from steal/steal) are redistributed equally among remaining players before next round
  • Need $4+ to qualify for charity donation
  • Odd player out auto-advances (never same person twice)
  • Game ends when too few players remain
