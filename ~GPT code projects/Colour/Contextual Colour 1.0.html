How to Play

Goal: Determine if all inner squares have the same color or if one differs.

The effect: Surrounding colors influence perception. Identical inner colors may appear different due to "simultaneous contrast."

Controls:

• Settings (⚙): Adjust color difference difficulty

• Levels (2/3/4): Change number of blocks

• JND settings: Very Narrow (20) to Very Wide (100)

How to play:

• Focus on inner squares only

• Click "Same" if all match, "Different" if any vary

• Outer colors fade to reveal the answer
