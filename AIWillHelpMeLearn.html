      How to share what Claude builds
      
    
      
      
      Claude builds you a single HTML file, or a chat tool. Here is how to actually use or share what it makes, easiest first. You do not need your own website.
      
      
        
        
      1. Use a platform you already have. For quizzes and flashcards, the export tools here give you a file or script to import into Quizlet, Anki, Blooket, Wayground or Google Forms, which then host the activity for you, with a normal link and no account for whoever uses it. Best for question-and-answer content.
        2. 
        
      2. Publish it as a Claude artifact. In Claude, open the tool it built, click Publish, and copy the link. Anyone opens it in a browser with no account and it runs. Catch: a published artifact cannot save progress, so spaced-repetition schedules reset, and a school-managed Team or Enterprise Claude account cannot publish publicly, use a personal Free, Pro or Max account.
        3. 
        
      3. Drop the file on Tiiny.host. Go to tiiny.host, drag the .html in, and get a link. This is the option that keeps saved progress, because it is a real web address. Catch: make a free account and log in about monthly to keep the link alive.
        4. 
        
      4. Google Sites (Workspace schools). New Site, then Insert, Embed, Embed code, paste the whole file, Publish, and share the link. It runs on sites.google.com, which school filters allow. Catch: very large files may be rejected by the embed box.
        5. 
        
      5. Send the file itself through Google Classroom or email as a last resort when sites are blocked. Catch: it will not run on iPhone or iPad, and there is no link, give phone users a link instead.
        6. 
      
      
      
      Saved progress (spaced repetition) only survives from a real web link such as Tiiny.host or Google Sites, not from a published artifact and not reliably from a downloaded file, and it never syncs between devices. Google Drive cannot run the tool, it only shows the code or downloads it. Test your link on a student device before you rely on it, school networks block some sites.
