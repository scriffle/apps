How it works

This tool decodes your audio, runs an FFT analysis, and turns it into a stream of discrete musical tokens — pitched notes, kicks/snares/hi-hats, beats, and section changes — then renders one visual event per token.

  • Renderers: a 2D Canvas / WebGL event-space, or a rotating 3D Cube point-cloud.
  • ≣ Tokens opens the controls — axis mapping, colour palette, motion, gravity, sources & sensitivity.
  • Cube: drag to orbit, scroll to zoom; map any feature to X / Y / Z.
  • ⤓ Save keeps the audio + tokens + settings as a project in this browser; ☰ Open reloads one.
