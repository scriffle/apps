Newton’s Laws of Motion — 1A Overview

  • Three laws: Inertia (1st), F = ma (2nd), Action–Reaction (3rd).
  • Forces add as vectors → the net (unbalanced) force sets the acceleration.
  • Free‑body diagrams help identify forces and directions.
  • Common contexts: motion at constant velocity, braking, vertical throws, inclined planes, pulleys & block systems, lifts.
Exam idea: Distinguish between balanced (ΣF = 0 ⇒ a = 0) and unbalanced forces (ΣF ≠ 0 ⇒ a ≠ 0).

Key formula set

  • a = Fnet / m  and  Fnet = m a
  • On slope: N = m g cos θ; Fds = m g sin θ; (μ = 0) ⇒ a = g sin θ
  • Elevator normal: N = m (avertical + g)
  • g ≈ 9.8 N·kg⁻¹ = 9.8 m·s⁻²

Summarised for learning from the user‑provided text (Cambridge 2023). For classroom use.

Newton’s First Law — Inertia

  • Statement: An object at rest or moving at constant velocity stays that way unless acted on by a net external force.
  • Inertia = resistance to change in motion; depends only on mass.
  • Balanced forces ⇒ ΣF = 0 ⇒ a = 0 (can be at rest or constant velocity).
  • Daily life feels counter‑intuitive because friction usually makes ΣF ≠ 0 → motion slows.
Examples: parked car (N balances weight); car cruising straight (drive balances drag/rolling resistance).

Newton’s Second Law — How forces change motion

  • Statement: a ∝ Fnet and a ∝ 1/m (same direction as Fnet).
  • a = Fnet / m   (units: N = kg·m·s⁻²)
  • Net force = vector sum of all forces. Use a free‑body diagram (FBD).

Method (always):

  1. Choose axes and a sign convention (write it down!).
  2. Draw FBD with magnitudes/directions.
  3. Resolve angled forces into components.
  4. Add components → Fnet; apply a = Fnet / m.

Worked Example — Aircraft thrust vs drag

A 66 000 kg aircraft in level flight: thrust 350 kN forward; resistance 300 kN backward.

Reveal worked answer
  • Fnet = 350 − 300 = 50 kN (forward)
  • a = Fnet / m = (50 000 N) / (66 000 kg) = 0.756 m·s⁻² forward
  • Vertical forces (lift & weight) balance in level flight.

Resolving forces — pull at an angle

3 kg block pulled by 30 N at 27° to horizontal; 20 N friction opposes motion.

Reveal worked answer
  • Horizontal pull = 30 cos 27° = 26.73 N
  • Fnet (horizontal) = 26.73 − 20 = 6.73 N
  • a = 6.73 / 3 = 2.24 m·s⁻²
Tip: Resolve angled forces before summing. Only horizontal components change horizontal motion.

Same acceleration ≠ same force

  • At Earth’s surface (ignore air), all masses fall with a = g = 9.8 m·s⁻².
  • But forces differ because F = m g:
    • 1 kg → 9.8 N; 2 kg → 19.6 N; 3 kg → 29.4 N.
  • a and v can point in different directions:
    • Braking car: v forward, a backward.
    • Ball thrown up: v up but a down (slows up, stops, then speeds down).

Inclined planes (θ)

  • Normal: N = m g cos θ
  • Down‑slope component: Fds = m g sin θ
  • Frictionless acceleration: a = g sin θ

Example (15 kg at 34°)

Reveal worked answer
  • N = (15)(9.8) cos 34° = 122 N
  • Fds = (15)(9.8) sin 34° = 82.2 N
  • Measured a = 0.397 m·s⁻² ⇒ friction opposes motion.
  • Ffriction = Fds − m a = 82.2 − (15)(0.397) = 76.2 N

Newton’s Third Law — Action–Reaction pairs

  • Statement: For every action force, there’s an equal and opposite reaction force.
  • Pairs act on different bodies (always).
  • Examples:
    • Jet exhaust pushes gas back ⇄ gas pushes jet forward.
    • Tyre pushes road back ⇄ road pushes tyre forward (driving force).
    • Free‑fall: Earth pulls you down ⇄ you pull Earth up.
  • Caution: Weight and normal on the same object are not an action–reaction pair.

Vertical motion & the normal force (lifts)

  • N = m (avertical + g)
  • At rest / constant velocity: avertical = 0 ⇒ N = mg.
  • Accelerate up ⇒ N > mg (“heavier”); accelerate down ⇒ N < mg (“lighter”).

Example (72 kg on scales)

Reveal worked answer
  • At rest: N = 706 N
  • 0.50 m·s⁻¹ constant down: N = 706 N
  • Upward a = 0.15 m·s⁻² ⇒ N = 716 N
  • Downward a = 0.30 m·s⁻² ⇒ N = 684 N

Applying Newton’s Laws to systems

Blocks pushed together

Push of 150 N on two blocks (5 kg & 2 kg) on frictionless surface.

Reveal worked answer
  • a = Fnet / (mA + mB) = 150 / 7 = 21.4 m·s⁻²
  • Contact force on B by A: F = mB a = 2 × 21.4 = 42.8 N → right
  • On A by B: 42.8 N → left (equal and opposite).

Pulley (table + hanging mass)

10 kg on table, 4.5 kg hanging; frictionless pulley & table.

Reveal worked answer
  • Fpull = mh g = 4.5 × 9.8 = 44.1 N
  • a = 44.1 / (10 + 4.5) = 3.04 m·s⁻²
  • String tension on 10 kg block: T = m a = 10 × 3.04 = 30.4 N
Strategy: Find system acceleration first (treat as one mass), then isolate parts to get internal forces (contact, tension, couplings).

Skill: directions, signs, and consistency

  • Pick a positive direction (e.g., “up is +”), write it down, and stay consistent.
  • Add forces with signs before applying a = Fnet / m.

Mini‑example (rocket)

  • Let up be +: Fnet = +650 + (−225) = +425 N ⇒ up
  • Or up be −: Fnet = −650 + 225 = −425 N ⇒ still “up” (by sign)

Quick check‑in

  1. Bowling ball slides right at constant v (no friction). Draw/label forces.
  2. Speedboat: m = 3600 kg, thrust 10 800 N, resistance 3500 N. Find a.
  3. Aircraft in horizontal flight: action = weight on aircraft. What’s the reaction?
Reveal answers
  • Forces: weight down, normal up (equal; ΣF = 0). No horizontal forces if “no friction”.
  • Fnet = 10 800 − 3500 = 7300 N ⇒ a = 7300 / 3600 = 2.03 m·s⁻² forward.
  • Earth pulled upward by the aircraft (gravitational attraction on Earth by aircraft).
